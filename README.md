# Recover NightOwl SVR Password
This script recovers the admin password for the NightOwl F6-series of devices, using the documented master password ```2x8axc```. The master password is useful for regaining access to the device, but in order to reset the actual admin password, or make any system changes, you need a valid admin-level account. This script *should* get that for you, otherwise you can capture network packets while you visit the Users section to get the password.

<img width="759" alt="nightOwl" src="https://user-images.githubusercontent.com/1743650/198940392-7006054c-d45f-4fd4-a137-f12215b90370.png">

Other possible passwords include:

```
519070
101101
```

For more information, see: https://web.archive.org/web/20200810153724/https://support.nightowlsp.com/hc/en-us/articles/360009216554-Night-Owl-Legacy-Devices
